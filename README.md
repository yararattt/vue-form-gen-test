# Vue 3 + TypeScript + Vite

Динамический генератор форм на Vue 3 с поддержкой JSON-схем и валидацией. Проект создан в рамках тестового задания.

## Установка и запуск

```bash
# Клонирование репозитория
git clone https://github.com/yararattt/vue-form-gen-test.git
cd vue-form-gen-test

# Установка зависимостей
npm install

# Запуск в режиме разработки
npm run dev

# Сборка для production
npm run build

# Превью сборки
npm run preview
